# 📈 Options Scalper Pro

**Options Scalper Pro** is a high-performance desktop application tailored for options scalpers in the Indian stock market. Built using the Zerodha Kite API, it empowers active day traders with fast, intuitive, and feature-rich tools for executing trades and managing positions with speed and confidence.

---

## 🚀 Key Features

### 🔁 Live & Paper Trading
- Seamlessly switch between **live trading** (via Zerodha Kite) and a **Paper trading mode** to test strategies without risk.

### 📊 Dynamic Strike Ladder
- Automatically centers on the **At-The-Money (ATM)** strike.
- Real-time updates for **LTP**, **Open Interest**, and **Bid/Ask spreads**.
- Ultra-responsive for rapid market analysis.

### ⚡ Rapid Order Placement
- **Buy Panel**: Select and place multiple strike orders at once.
- **Quick Order Dialog**: Instantly place or modify orders on a single contract.

### 📋 Position Management
- View and manage your portfolio in real time.
- Track **Unrealized** and **Realized P&L** live.
- Exit all positions, or just **CALLs** or **PUTs**, with a single click.

### 🕒 Persistent History & Analytics
- All trades are logged locally and viewable in an **Order History** dialog.
- View daily performance in a **P&L History Calendar**.

### 🔐 Secure Credentials
- Zerodha API keys are securely encrypted and stored on your local machine.

---

## 🛠️ Technology Stack

- **Language**: Python
- **GUI Framework**: PySide6 (Qt for Python)
- **API**: Zerodha Kite Connect
